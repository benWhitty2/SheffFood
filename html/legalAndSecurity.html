Legal and Security

computer misuse act

the contact information given in on the contact page does not relate to a real email or phone no, this is to remove the risk of targeted phishing attacks. Comments made about the site would be stored on an email server so should be secure if a trustworthy provider is used. The only other security concern is how vulnerable the web server is to ddos attacks, because of the large number of images used in it, however this is an issue that would need to be resolved on the backend.

general data protection act

any information that the users input through the contact form will be emailed exclusively to a secure email with a strong password only known by appropriately authorized people, that will be read regularly. Emails will be deleted as soon as they are no longer relevant to avoid holding personal data and sender's emails will be removed with them. Data held and displayed about vendors is non-personal data so this act does not apply to it

copyrights designs and patents act

This website is only for internal use inside the university and will never be deployed; this means that images used on it will not be subject to copyright laws as they will not be used publicly. All information about vendors used in this website is public information so also not affected by these laws
